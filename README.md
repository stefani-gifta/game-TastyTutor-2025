# Tasty Tutor

**Tasty Tutor** is a simple and fun cooking game, made for all ages!

## Game Features

* Choose from five recipes, each with a video answer key
* Drag-and-drop cooking mechanics
* Cute animated character that reacts to your performance
* Timer-based challenge for extra excitement
* Emoji confetti and sound effects

## Play Now!

[Click here to start playing!](https://stefani-gifta.github.io/game-TastyTutor-2025/html/spa.html)

## How to Run Locally

1. Clone the repository:

   ```bash
   git clone https://github.com/stefani-gifta/game-TastyTutor-2025.git
   ```
2. Open `html/spa.html` in your browser.
